# Live Cricket Match Tracker

A web application that fetches and displays live cricket matches and detailed scorecards using real-time data from the Cricbuzz RapidAPI.

Note: API requests are limited under the free RapidAPI plan.

---

## Features

- View all currently live cricket matches
- Click a match to open the full scorecard
- Display batting and bowling statistics for both teams
- Player-level stats including runs, balls, fours, sixes, overs, wickets, etc.
- Manual refresh option to update match data

---

## Tech Stack

HTML • CSS • JavaScript • Cricbuzz RapidAPI
